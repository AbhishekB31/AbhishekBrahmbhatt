Key Features


  • Thrilling 2D platform gameplay in a picturesque snowy mountain environment.
  • Added particle effects for an immersive skiing experience.
  • Speed control with the up and down arrow keys for precision skiing.
  • Perform impressive backflips and frontflips to show off your skills.
  • Quick respawn from the starting point upon unfortunate accidents, ensuring non-stop action and fun.
Ski Buddy is a thrilling 2D platform game set in a stunning snowy mountain landscape, featuring added particle effects for an immersive experience. Players can control their speed with the arrow keys, execute flips, and enjoy quick respawns for uninterrupted skiing excitement.
